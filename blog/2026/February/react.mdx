# React 入门实战

React 是目前主流的前端 UI 框架，采用组件化思想开发，具备高效、灵活、易复用的特点。本文带你从零入门 React。

## 一、环境准备
使用官方脚手架快速创建 React 项目：
```bash
# 安装脚手架
npm install -g create-react-app

# 创建项目
create-react-app my-react-demo

# 启动项目
cd my-react-demo
npm start
