# Node.js + Express 快速搭建后端接口

对于前端开发者而言，使用 Node.js 可以快速搭建简易后端，实现前后端联调。本文使用 Express 框架演示接口开发。

```bash
# 新建文件夹

# 初始化包管理

# 安装依赖
