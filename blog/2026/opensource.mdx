# 我的开源项目：MonoBlog 个人博客

本文介绍我搭建的个人博客开源项目，基于 Docusaurus 静态站点框架 + Supabase 后端服务。

## 一、项目功能
本博客完整实现以下功能：
- 静态页面部署，支持 GitHub Pages 免费托管
- 邮箱 / GitHub 双渠道账号登录
- 侧边栏实时评论区
- 在线用户数量、最新注册用户实时统计
- 全站移动端自适应布局
- 文章分类、标签、导航跳转

## 二、技术栈
- **前端框架**：Docusaurus（基于 React）
- **样式/布局**：原生 CSS + Flex 弹性布局
- **后端服务**：Supabase（BaaS 云服务，提供认证、数据库）
- **部署平台**：GitHub Pages

## 三、开源地址
项目已开源，欢迎 Star、Fork、提 Issue 交流：
> 地址：https://github.com/ye2f4/pblot

## 四、使用建议
1. 克隆代码到本地，安装依赖后即可本地运行调试；
2. 自行配置 Supabase 密钥、站点域名、路由信息；
3. 支持自定义样式、导航、文章内容，可二次开发。

## 总结
该项目适合想搭建个人技术博客、学习 Docusaurus + Supabase 组合方案的开发者，欢迎大家一起交流学习。
