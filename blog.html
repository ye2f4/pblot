# 我的开源项目：MonoBlog 个人博客

本文介绍我搭建的个人博客开源项目，基于 Docusaurus 静态站点框架 + Supabase 后端服务。

## 一、项目功能
本博客完整实现以下功能：
- 静态页面部署，支持 GitHub Pages 免费托管
- 邮箱 / GitHub 双渠道账号登录
- 侧边栏实时评论区
- 在线用户数量、最新注册用户实时统计
- 全站移动端自适应布局
- 文章分类、标签、导航跳转

## 二、技术栈
- **前端框架**：Docusaurus（基于 React）
- **样式/布局**：原生 CSS + Flex 弹性布局
- **后端服务**：Supabase（BaaS 云服务，提供认证、数据库）
- **部署平台**：GitHub Pages

## 三、开源地址
项目已开源，欢迎 Star、Fork、提 Issue 交流：
> 地址：https://github.com/ye2f4/pblot

## 四、使用建议
1. 克隆代码到本地，安装依赖后即可本地运行调试；
2. 自行配置 Supabase 密钥、站点域名、路由信息；
3. 支持自定义样式、导航、文章内容，可二次开发。

## 总结
该项目适合想搭建个人技术博客、学习 Docusaurus + Supabase 组合方案的开发者，欢迎大家一起交流学习。

# 提升开发效率的 VSCode 常用插件

VSCode 是主流代码编辑器，搭配插件可以极大提升开发效率，下面分享我日常必备的插件。

## 1. Prettier
代码自动格式化工具，支持 JS/CSS/MDX 等几乎所有文件，保存文件即可自动统一代码风格。

## 2. ESLint
代码语法检查，排查语法错误、不规范写法，团队协作必备。

## 3. GitLens
Git 增强插件，在行内直接查看代码提交记录、作者、时间，追溯代码变更非常方便。

## 4. Auto Rename Tag
自动同步修改 HTML/JSX 标签，修改开始标签时，闭合标签自动同步修改。

## 5. Path Intellisense
文件路径自动补全，导入模块、引用资源时自动提示路径，减少手动输入错误。

## 6. REST Client
在编辑器内直接测试 HTTP 接口，无需额外安装 Postman。

## 7. Markdown All in One
Markdown/MDX 增强插件，优化文档编写、预览、表格/列表快捷操作。

## 总结
以上插件覆盖了编码、格式化、版本管理、接口测试等全流程，推荐前端/全栈开发者安装使用。

  2. 下载对应游戏版本的 Forge MDK（模组开发包）
  3. 使用 IDEA / Eclipse 导入开发工程

模组入口主类，负责初始化、注册各类游戏内容：

import net.minecraftforge.fml.common.Mod;

@Mod("demo_mod")
public class DemoMod {
    public static final String MOD_ID = "demo_mod";

    public DemoMod() {
        // 执行注册、初始化逻辑
    }
}
